# FdE 2026 — StockIA

Presentación interactiva del proyecto **StockIA**: un SaaS de gestión de inventario para
PyMEs argentinas potenciado por IA/OCR. Trabajo para la materia **Formación de Emprendedores**
(Equipo 1, 2026).

🔗 **Ver online:** https://samuop.github.io/FdE-2026/

## Contenido (12 slides)

1. Portada
2. Paso 1 · Idea de negocio
3. Paso 2 · Segmento de clientes
4. Paso 3 · Mapa de empatía
5. Paso 4 · Early Adopters
6. Paso 5 · Lienzo de Propuesta de Valor
7. Paso 6 · Análisis de competencia
8. Paso 7 · Business Model Canvas (9 bloques)
9. Paso 8 · Ciclo Construir–Medir–Aprender
10. Paso 9 · Presentación final (pitch)
11. Business Model Canvas (lienzo completo)
12. Conclusiones

## Navegación

- **← →** o **barra espaciadora** para avanzar / retroceder
- **Clic** en la mitad derecha/izquierda de la pantalla
- **Botones** y **puntos** de la barra inferior
- **Home / End** para ir al inicio / final

## Tecnologías

- [Vite](https://vite.dev/) + [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Framer Motion](https://www.framer.com/motion/) para las animaciones
- [Lucide](https://lucide.dev/) para la iconografía

## Desarrollo

```bash
npm install
npm run dev      # servidor de desarrollo
npm run build    # build de producción a /dist
npm run preview  # previsualizar el build
```

## Despliegue

El sitio se despliega automáticamente a **GitHub Pages** con GitHub Actions
en cada push a `main` (ver `.github/workflows/deploy.yml`).
